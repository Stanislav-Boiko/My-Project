Теоретичні питання: 1. function name(params) { }; name() ; 2. Оператор return в JavaScript використовується для повернення значення з функції. Може бути використаний для повернення будь-якого типу даних 3. Параметри - це змінні, які оголошені у визначенні функції. Параметри визначають, які дані можуть бути передані у функцію. Аргументи - це фактичні дані, які передаються при виклику функції. Аргументи використовуються для передачі даних з місця виклику функції до самої функції. 4. Просто передати її як аргумент в іншу функцію. (callback)
